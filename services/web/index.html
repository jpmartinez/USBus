Bienvenido a la Api de USBUS

A continuacion se detallan las URIS de los servicios brindados por USBUS

  • Registro
  • HOST:IP/rest/api/register/tenant POST
  • HOST:IP/rest/api/register/user POST
  • Autenticación
  • HOST:IP/rest/api/authentication POST
  • Administración
  • HOST:IP/rest/api/{tenantId}/busstop/{busStopId} GET,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/branch/{branchId} GET?query:{NAME,ALL}|offset|limit|branchName,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/route/{routeId} GET?query:{ORIGIN,DESTINATION,ORIGIN_DESTINATION,ALL}|origin|destination|offset|limit,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/service/{serviceId} GET?query:{DAYOFWEEK,DAYOFWEEK_FROM_TO,ALL}|origin|destination|dayOfWeek|offset|limit,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/users/{username} GET?query:{ALL}|offset|limit,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/hr/{username} GET?query:{ALL,STATUS,HRSTATUS,EMAIL}|email|status|hrstatus|offset|limit,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/ticket/{serviceId} GETusername|status|journeyId|offset|limit,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/journey/{journeyId} GET?query:{STATUS,DATE,DATE_STATUS,DATE_ORIGIN_DESTINATION}|origin|destination|status|date|offset|limit,POST,PUT,DELETE
  • HOST:IP/rest/api/{tenantId}/journey/{journeyId}/price GET
  • HOST:IP/rest/api/{tenantId}/bus/{busId} GETbusStatus|offset|limit,POST,PUT,DELETE
