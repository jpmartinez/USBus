Bienvenido a la Api de USBUS

A continuacion se detallan las URIS de los servicios brindados por USBUS

  • Registro
  • HOST:IP/rest/api/register/tenant POST
  • HOST:IP/rest/api/register/user POST
  • Autenticación
  • HOST:IP/rest/api/authentication POST
  • Administración
  • HOST:IP/rest/api/{tenantId}/busstop/{busStopId} GET,POST,PUT,DELETE
